# Noticed

ひとりじゃないと思える瞬間を、どこかにいる誰かと。

You're not alone. Someone just noticed.

## About

Noticed is an anonymous loneliness signal app. One tap when you feel alone — an anonymous someone responds: "I'm here." No conversation. No DMs. Just being noticed.
